- It looks like netlify has 2 apps that they use - one for authentication and another for accessing the user's code for ci/cd. They can be viewed at this link: https://github.com/settings/applications.
- If you see `The request failed because it didn't meet the preconditions in one or more request-header fields.` error for github actions cloudfront lambda function upload, just re-run the github action. It's a problem with AWS.
